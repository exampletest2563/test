## HTML Headings

HTML headings are defined with the `<h1>` to `<h6>` tags.

```html
<!DOCTYPE html>
<html>
	<head>
	</head>
	<body>
		<h1>Main Heading</h1>
		<h2>Sub Heading 2</h2>
		<h3>Sub Heading 3</h3>
		<h4>Sub Heading 4</h4>
		<h5>Sub Heading 5</h5>
		<h6>Sub Heading 6</h6>
	</body>
</html>
```

`<h1>` is used for the most important heading. `<h6>` is used for the least important heading.

Headings are important.

## HTML Lists

HTML lists allow developers to group a set of related items.

- Unordered HTML List

```html
<!DOCTYPE html>
<html>
	<head>
	</head>
	<body>
		<ul>
			<li>Home</li>
			<li>Categories</li>
			<li>Contact Us</li>
			<li>FAQ</li>
		</ul>
	</body>
</html>
```

- Ordered HTML List

```html
<!DOCTYPE html>
<html>
	<head>
	</head>
	<body>
		<ol>
			<li>First Item</li>
			<li>Second Item</li>
			<li>Third Item</li>
			<li>Fourth Item</li>
		</ol>
	</body>
</html>
```
